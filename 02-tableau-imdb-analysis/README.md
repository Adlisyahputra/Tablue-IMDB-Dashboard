# IMDb Movies Analysis

## Objective
Analyze IMDb movie data to explore movie ratings, voting patterns, production trends, and geographic distribution.

## Tools
- Google Sheets (data cleaning)
- Tableau Public (data visualization)

## Dataset
Cleaned IMDb movies dataset containing information such as:
- Movie title
- Release year
- IMDb score
- Votes
- Country
- Director
- Duration
- Content rating

## Process
- Cleaned raw IMDb data using spreadsheet
- Removed duplicates and handled missing values
- Standardized column names and data types
- Built an interactive dashboard in Tableau

## Dashboard
https://public.tableau.com/shared/R8JFJMRH6?:display_count=n&:origin=viz_share_link

## Key Insights
- Movies with high IMDb ratings do not always receive the highest number of votes
- Movie production shows an increasing trend after the year 2000
- Drama and popular genres tend to have higher average ratings
- IMDb movies are dominated by productions from a few countries
